duoludo

Duoludo is a game whose purpose is games.

For game designers, duoludo is a tool for achieving high-coverage testing of games.

For game players, duoludo is a hub for competing with friends on casual bite-size game tasks.

ZEPTO is a example game that was developed in conjunction with duoludo. Click the links below to play ZEPTO through duoludo (Tested in Chrome and Firefox) or to view some of the data collected with duoludo.
